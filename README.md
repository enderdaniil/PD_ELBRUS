# Проект: «Разработка транслятора из формата ".orc" в ".c" с использованием SIMD функций»

В framework-е Gstremer есть шейдеры для работы с видео и звуком, описываются они в текстовом файле с расширение .orc. Суть задачи в том, что мы оптимизируем работу шейдеров с помощью SIMD-функций.


[Ссылка на сайт проекта](http://pd-2021-1.std-1048.ist.mospolytech.ru)
